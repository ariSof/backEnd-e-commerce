# E-commerce-backend
ORM backend app

## Description

Backend setup for an e-commerce site that will need to keep track of products, categories and product tags.

Video walkthrough: https://www.awesomescreenshot.com/video/24653844?key=c5f6f56ed06467625e10690b92bfe35b

## Installation

* Run schema- I used MySQL Workbench
* Update database connection information
* npm i 
* seed database by running: npm run seed

## Usage

Run node server.js in the command line. Then use postmates or Insomnia to test the different routes. Five routes are available for each categories, products, and tags. GET route to view all, GET route to view one by id, POST route to add a new (category, product, or tag) with included JSON data, PUT route to update an existing one by id, and DELETE route by id.
